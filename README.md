# Reloj TEA

Panel visual de comunicación emocional para niños con TEA.

Este repositorio contiene el panel HTML público del proyecto Reloj TEA.
El panel permite que un niño señale cómo se siente usando colores, caras y palabras simples.

## Qué es

Reloj TEA es un proyecto de apoyo a la comunicación emocional para niños con Trastorno del Espectro Autista.

La idea principal es sencilla:

Un niño puede tener dificultades para explicar lo que siente con palabras.
Un panel visual puede ayudarle a expresarlo con una acción más simple: pulsar una emoción.

## Panel visual

El panel incluye cinco estados:

| Estado    | Color    | Emoji | Uso                                           |
| --------- | -------- | ----- | --------------------------------------------- |
| Agobiado  | Rojo     | 😭    | Sobrecarga o malestar intenso                 |
| Nervioso  | Naranja  | 😰    | Ansiedad, inquietud o tensión                 |
| Triste    | Amarillo | 😢    | Tristeza o bajón emocional                    |
| Bloqueado | Azul     | 😶    | Bloqueo, silencio o dificultad para responder |
| Bien      | Verde    | 🙂    | Estado tranquilo o estable                    |

El panel está pensado para usarse con un adulto presente.

El niño señala una emoción.
El adulto recibe una orientación breve para acompañar mejor ese momento.

## Cómo funciona

El panel está hecho con HTML, CSS y JavaScript básico.

No usa cuentas.
No guarda datos.
No pide información personal.
No necesita instalación.

Funciona en móvil, tablet y ordenador.

## Uso dentro del GPT

Este panel acompaña a un GPT personalizado de ChatGPT.

El GPT guía al adulto.
El panel ayuda al niño a comunicar cómo está.

Flujo previsto:

1. El adulto abre el GPT.
2. El adulto abre el panel visual.
3. El niño pulsa una emoción.
4. El adulto indica al GPT qué emoción ha elegido el niño.
5. El GPT ofrece una guía práctica para acompañar la situación.
